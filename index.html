CSS Styling

What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages.

Why Do We Use CSS?

  • Change colors and fonts.
  • Improve page layout.
  • Create responsive designs.
  • Enhance user experience.

Example CSS Code


body {
    background-color: lightblue;
}

h1 {
    color: navy;
}

p {
    font-size: 18px;
}

Benefits in Smart Learning Platform

CSS helps make the learning platform attractive, organized, and easy to use on different devices.
